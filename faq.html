How do I get started with Voice Reader?

Getting started is easy:

  • Download Voice Reader from the App Store
  • Open the app and complete the onboarding
  • Set up a TTS provider (we recommend Azure Speech)
  • Import an EPUB book from the Files app
  • Tap play and enjoy!

Is Voice Reader free to use?

Yes, Voice Reader is free to download and use. However, you'll need to provide your own TTS API key. Azure Speech offers a free tier with 500,000 characters per month, which is plenty for most users.

What devices are supported?

Voice Reader requires iOS 17.0 or later. It works on iPhone and iPad.

How do I get an Azure API key?

Follow these steps:

  • Create a free Azure account at azure.microsoft.com
  • Create a Speech resource in the Azure portal
  • Copy your API key from Keys and Endpoint
  • Enter the key in Voice Reader settings

Can I use Voice Reader offline?

Yes, with the Piper TTS option. Piper is a self-hosted TTS service that runs on your local network. You can set up a Piper server on your computer or NAS and use Voice Reader without internet.

üîä Audio Issues

No audio is playing when I tap play

Try these troubleshooting steps:

  • Check your device volume and silent mode
  • Verify your TTS API credentials in Settings
  • Test your API key using the test button in the TTS wizard
  • Check your internet connection
  • Try a different voice in Settings

Text highlighting is not synchronized

Highlighting sync depends on several factors:

  • Try adjusting the speech rate in Settings
  • Some complex book layouts may affect sync accuracy
  • Ensure you're using a stable internet connection

Audio stops when I switch apps

Voice Reader supports background playback. Make sure:

  • Background App Refresh is enabled for Voice Reader
  • Low Power Mode is disabled (it can interrupt background audio)

How do I add bookmarks or highlights?

While reading:

  • Bookmark: Tap the bookmark icon in the toolbar
  • Highlight: Long-press on text to select, then tap "Highlight"
  • Note: Select text and tap "Add Note"

View all your annotations from the Notes panel.

Does Voice Reader have a sleep timer?

Yes! Tap the timer icon in the playback controls to set a sleep timer. Options include 5, 15, 30, 60 minutes, or end of chapter.

Can I change the reading theme?

Yes, Voice Reader supports multiple themes:

  • Light: White background
  • Sepia: Warm, paper-like background
  • Dark: Dark background for night reading

Tap the appearance icon (Aa) in the reader to change themes, font size, and more.

üìö Book Import

What file formats are supported?

Voice Reader currently supports EPUB format. This is the most common e-book format and is supported by most e-book stores.

PDF support may be added in a future update.

How do I import books?

There are several ways to import books:

  • Tap the + button in your library and select from Files
  • Open an EPUB file from Files app ‚Üí Share ‚Üí Voice Reader
  • Download an EPUB from Safari ‚Üí Open in Voice Reader

Can I import DRM-protected books?

No, Voice Reader only works with DRM-free EPUB files. Books purchased from most stores include DRM protection. You can find DRM-free books from:

  • Project Gutenberg (free public domain books)
  • Standard Ebooks (free, beautifully formatted classics)
  • Some publishers that sell DRM-free editions
