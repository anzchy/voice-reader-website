Piper TTS Setup

Self-hosted offline text-to-speech for complete privacy

ðŸ”’ Why Piper?

Piper is a self-hosted TTS solution that runs entirely on your local network. Your text never leaves your home, making it perfect for privacy-conscious users.

Requirements

  • A computer, NAS, or server to run Piper
  • Docker (recommended) or Python 3.9+
  • Your iOS device on the same network

Quick Start with Docker

The easiest way to run Piper is with Docker:

docker run -d \ --name piper \ -p 5000:5000 \ rhasspy/piper

Configure Voice Reader

In Voice Reader settings:

  • Provider: Piper (Offline)
  • API URL: http://YOUR_SERVER_IP:5000/api/tts
  • API Key: Leave empty (not required)

Available Voices

Piper supports many voices in different languages. Download voice models from the Piper GitHub repository.

For the best experience, run Piper on a device that stays on, like a NAS or Raspberry Pi. This ensures TTS is always available when you want to read.
