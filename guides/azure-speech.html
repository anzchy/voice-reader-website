Microsoft Azure Speech Setup

Get your Azure Speech API key and start using high-quality HD neural voices in Voice Reader.

üí∞

Free Tier Available

Azure Speech offers 500,000 characters free per month on the standard tier. That's approximately 5-10 hours of audiobook listening. No credit card required for the free tier.

1

Create an Azure Account

If you don't have an Azure account, create one for free:

  1. Go to azure.microsoft.com/free
  2. Click "Start free" or "Try Azure for free"
  3. Sign in with your Microsoft account or create a new one
  4. Complete the registration (phone verification required)

Note: You may need to provide a credit card for verification, but you won't be charged for the free tier usage.

2

Create a Speech Resource

Create an Azure Speech service resource:

  1. Go to the Azure Portal
  2. Click "Create a resource"
  3. Search for "Speech" and select "Speech" by Microsoft
  4. Click "Create"
  5. Fill in the required fields:
    • Subscription: Select your subscription
    • Resource group: Create new or select existing
    • Region: Choose one close to you (see table below)
    • Name: Give it a name (e.g., "voice-reader-speech")
    • Pricing tier: Select "Free F0" or "Standard S0"
  6. Click "Review + create", then "Create"
Screenshot: Azure Speech resource creation form
3

Get Your API Key

Once your resource is created:

  1. Go to your Speech resource in the Azure Portal
  2. Click "Keys and Endpoint" in the left menu
  3. Copy KEY 1 (or KEY 2)
  4. Note your Location/Region (e.g., "eastus")
Screenshot: Keys and Endpoint page

‚úì You now have what you need!

Keep your API key safe. You'll enter it in Voice Reader settings.

4

Configure Voice Reader

Open Voice Reader and go to Settings:

  1. Tap the gear icon to open Settings
  2. Select "Microsoft Speech Foundry" as the provider
  3. Enter the API URL using your region:
    https://{region}.tts.speech.microsoft.com/cognitiveservices/v1
  4. Paste your API Key
  5. Select a voice (Andrew HD recommended)
  6. Tap "Test" to verify it works

Example URL for East US 2:

https://eastus2.tts.speech.microsoft.com/cognitiveservices/v1
